# Sea Attitudes — Product Photography Prompt Pack

Studio-quality photo prompts for **every** piece, ready to paste into ChatGPT (or any image model that accepts an image input).

## How to use it (the 30-second version)

1. Open ChatGPT with image generation.
2. **Attach the real reference photo** for the piece — this keeps the artwork true. The file is named under each piece (e.g. `IMG_8955.jpg`) and lives in `public/products/`.
3. Paste the **Studio (hero)** prompt and press go.
4. For more angles, paste the **Lifestyle** or **Macro** prompt.
5. Download it, **save with the exact filename shown** (e.g. `IMG_8955.jpg`), and drop it into `public/products/`, replacing the old file. The site updates automatically — no code changes.

> **Looks like a real studio photo, not AI:** always attach the real photo first, and keep the original if a result alters any shells or glass. The prompts already specify true photographic realism (camera, softbox lighting, real shadows).

> **Want to automate it?** If you have an OpenAI API key, run `node scripts/generate-studio-photos.mjs` — it sends each real photo + its Studio prompt to the image API and writes the results for you. See `scripts/README-photos.md`.

## The 7 catalog renditions

| # | Shot | Size | Use |
|---|------|------|-----|
| 1 | Hero / Studio | 3000×3000 | Product page + grids |
| 2 | Lifestyle (room) | 3840×2560 | Banners, collections |
| 3 | Editorial portrait | 2400×3600 | Blog, features |
| 4 | Macro detail | 2400×2400 | Craftsmanship close-ups |
| 5 | Transparent PNG | 3000×3000 | Compositing |
| 6 | Mobile hero crop | 1440×2560 | Mobile headers |
| 7 | Thumbnail | 800×800 | Fast grids |

Add to any prompt to change the rendition, e.g. *“Instead, output a transparent-background PNG, 3000×3000, the piece isolated with clean edges.”*
